# Food Delivery Data Analysis (SQL + Python + Power BI)

## 📌 Project Overview
This project performs end-to-end analysis of a food delivery dataset using SQL, Python, and Power BI.  
The goal is to analyze revenue trends, customer behavior, and operational efficiency.

---

## 🛠️ Tools Used
- Python (Pandas, Data Cleaning)
- SQL (Data Analysis)
- Power BI (Dashboard Visualization)

---

## 📊 Key Metrics
- Total Revenue: 26.89M
- Total Orders: 100K
- Average Order Value: 268.93
- Average Delivery Time: 37.52 minutes

---

## 🔍 Key Insights
- Revenue is evenly distributed across food items, with Shawarma contributing ~11.38%
- No single restaurant dominates revenue, indicating a competitive marketplace
- Orders remain stable across the 16-day period
- Peak order hours highlight demand concentration at specific times

---

## 💡 Business Recommendations
- Promote top-performing items like Shawarma to increase revenue share
- Introduce product differentiation to increase average order value
- Optimize delivery operations during peak hours

---

- `data/` → Dataset
